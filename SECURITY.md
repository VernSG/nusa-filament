# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of `vernsg/nusa-filament`.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report security concerns privately through the repository security advisory feature on GitHub. If that is unavailable, contact the maintainer privately and include:

- a clear description of the issue;
- affected versions;
- reproduction steps;
- possible impact;
- any suggested fix, if available.

The maintainer will review the report and coordinate a fix before public disclosure when appropriate.
