# Changelog

All notable changes to `vernsg/nusa-filament` will be documented in this file.

This project follows semantic versioning where possible.

## Unreleased

## 0.1.1

- Improved dependent select UX by preloading options for the selected parent region.
- Added default option loading while keeping remote search support.

## 0.1.0

- Initial release.
- Added Filament form components for province, regency, district, and village selects.
- Added `NusaAddress` form group.
- Added table columns and location filter.
- Added infolist components.
- Added region code and hierarchy validation rules.
